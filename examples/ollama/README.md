# Ollama

## Install


```bash
pip install langchain-ollama
```

```bash
# On macOS
brew install ollama
```

```bash
# On Linux
curl -fsSL https://ollama.com/install.sh | sh
```

```bash
# Run ollama
ollama serve
```

```bash
# Pull model
ollama pull qwen2.5:32b
```
